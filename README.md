# 모드의 첫 등장
마참내 모드가 등장했어요!
패치 노트는 아래에 있으니 참고 부탁드립니다!

1. 핵심 패치
-자원이라는 큰 제약을 없에버린 **연금술**을 완전히 삭제했습니다. 
모드 개발진은 연금술이 멀티플레이에 큰 영향을 끼치며 이로 인해 밴당한 국가가 생길정도로 문제가 크다고 판단하여 제거하게 되었습니다.
그래도 있을수 있으니 버그 제보 환영합니다.

2. 이집트 추가
-원역사 대로라면 존재해야할 이집트가 패독의 멍청한 짓으로 인해 존재하지 않게 되었습니다. 멀티플레이 모드에서 이집트를 부활시켰으며 그리 효과가 좋진 않을것입니다.

3. 랜덤박스 추가
-예전 구 멀티서버에서도 논의되었던 기능중 하나인 랜덤박스가 드디어 공식적으로 출시되었습니다. 사용방법은 게임 규칙 설정에서 제일 밑에 보시면 랜덤박스가 있을겁니다.

4. 버그 수정
-독소전이 발발할때 핀란드와 소련이 전쟁을 하지 않았더라도 핀란드가 독일편에서 참전하는 버그를 수정하였습니다.
4-1.  
-에티오피아가 영국으로 망명을 가고 프랑스가 연합국에 가입할 시 에티오피아편으로 참전할 수 있던 버그를 수정했습니다.
4-2. 
-영국이 일부 국가를 **해방이 가능한** 버그를 수정했습니다.
4-3.  
-스웨덴이 멀티 유저수 만큼 민병대를 복사하던 버그를 수정하고 무조건 20개로 고정했습니다.
4-4.  
-암트랙(수륙양용장갑차) 설계사 버그를 수정하였습니다.
4-5.  
-그 외 기타 버그들을 수정했습니다.

5. 버프/너프  
-상대적으로 강해 게임 벨런스를 붕괴시켰던 근접항공지원기의 성능을 대폭 하향시켰습니다.
5-1.  
-기갑 대대의 조직력을 7.5% 상승시켰습니다.
5-2.  
-구축전차의 조직력을 약 15%정도 감소시키고 생산비용을 조금 늘렸으며 대물과 관통은 유지했습니다.
5-3.  
-시암 연구속도 버프를 대폭 하향시켰습니다.

6. 기타
-많은 국가들이 사라졌습니다. 최적화를 위한 작업이니 어쩔수 없습니다.
```
중남미 -> 통합
에티오피아 분할 -> X
예멘, 오만, 팔레스타인, 요르단 -> 영국
시리아, 레바논 -> 프랑스
네팔, 부탄, 버마 -> 영국령 인도
탄누투바, 몽골 -> 소련
중점이 없는 국가들을 위주로 삭제했습니다.
```
6-1.  
-게임 최적화를 위해 공산 영국 루트 중 식민지을 해방하는 대신 순응도와 저항도를 좀 높이기로 결정했습니다.
6-2.  
-지속적 중점들을 일부 추가하였습니다.
6-3.  
-형성가능국가를 몇 개 추가하였습니다.
6-4.  
-프랑스는 알제리, 라오스, 캄보디아 제외 나머지는 해방 불가로 변경되었습니다.
